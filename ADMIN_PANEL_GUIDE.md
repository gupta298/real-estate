# Off-Market Deals Admin Panel Guide

This guide explains how to manage off-market deals using the Admin Panel.

## Access

1. Navigate to `/admin/off-market-deals` (requires admin login)
2. Click "Create New Deal" to add a listing
3. Click "Edit" on any deal to modify it

## Features

### Creating/Editing Deals

The admin panel provides a user-friendly interface with the following sections:

#### Basic Information
- **Title** (Required): Deal title
- **Content** (Required): Full description (supports emojis, line breaks, formatting)

#### Property Details
- **Property Type**: Select from "Home" or "Business"
- **Sub-Type**: 
  - For Home: Single Family, Townhouse, Condo, Multi-Family
  - For Business: Alcohol Business, Gas Station, Restaurant, Retail, Office
- **Area/Location**: General location (e.g., "Indianapolis, IN")
- **Status**: Open, Pending, or Closed

#### Contact Information
- **Contact Name**: Person's name
- **Contact Title**: Job title (e.g., "CEO / Broker")
- **Contact Phone**: Phone number
- **Contact Email**: Email address

#### Images
- Add multiple image URLs (one at a time)
- First image will be the primary/thumbnail image
- Images can be removed by clicking the X button
- Use publicly accessible image URLs

#### Settings
- **Active**: Toggle to show/hide the deal on the public site

- **Display Order**: Number to control listing order (0 = first)

1. **Titles**: Use clear, descriptive titles
2. **Content**: Include emojis for visual appeal, use line breaks for readability

## Tips

- **Image URLs**: Make sure image URLs are publicly accessible
- **Content Formatting**: Line breaks and emojis are preserved
- **Bulk Updates**: Edit deals one at a time through the admin panel
- **Preview**: Changes are visible immediately after saving

## Troubleshooting

- **Can't access**: Make sure you're logged in as admin
- **Images not showing**: Verify image URLs are publicly accessible
- **Changes not saving**: Check browser console for errors
- **Missing fields**: All fields are optional except Title and Content
